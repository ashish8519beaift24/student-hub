# 🎓 Student Hub 📊

Student Hub is a comprehensive web application designed to help students track their academic progress and predict their future performance using Machine Learning.

## 🚀 Features

- 🧮 **GPA Calculator**: Calculate your Grade Point Average (GPA) based on your course grades and credits.
- 📈 **Score Predictor**: Predict your exam score based on study hours and attendance using a trained Machine Learning model.

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Python 🐍 | Core Development |
| Flask | Web Framework |
| Pandas & NumPy | Data Handling & Computation |
| Scikit-learn | Machine Learning Model (`study_model.pkl`) |
| HTML/CSS | Frontend Interface |

## ⚙️ Installation & Setup

1. **Install Dependencies**
   ```bash
   pip install flask numpy pandas scikit-learn joblib
   ```

2. **Run the Project**
   ```bash
   python app.py
   ```

3. **Open in Browser**
   Navigate to `http://127.0.0.1:5000/`
